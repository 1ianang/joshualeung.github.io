Introducing RedTrans - The Missing Translation Tool for REDNote

2025-01-13

RedTrans is a dedicated Chrome extension that allows you to translate text in REDNote (also known as Xiaohongshu, or 小红书) website (https://www.xiaohongshu.com/).

Features

  • Translate titles on explore page, title/content/comments on note detail page, message on notification page
  • Use LLM to translate text, OpenAI and DeepSeek supported as of now.
  • Minimal, no ads, no tracking, no data collection.

How to use

Install

  1. Download the zip file from https://orrrrz.github.io/downloads/RedTrans-v0.1.zip and unzip it.(Alternatively, you can clone the github repo: https://github.com/orrrrz/red-trans)
  2. Open Chrome and go to chrome://extensions/ and swith to Developer mode.
  3. Click “Load unpacked”
  4. Select the folder you unzipped
  5. The options page will be opened automatically, you can set your API key there. You can get the API key from OpenAI/DeepSeek website.
  6. Done!

Usage

Hove on the title/comments/notifications, the translate button will show.

Have Fun!

Common Questions

How should I get the API key?

  • For DeepSeek, head to https://platform.deepseek.com/. Register first if you haven’t. Then from the left menu, click “API Keys” and click the “Generate API Key” button.

  • For OpenAI, head to https://platform.openai.com/settings/organization/api-keys. Register first if you haven’t. Click the “Create new secret key” button.

The translate button doesn’t show up

  • Refresh the page since the button has a probability to fail to load.
  • Otherwise it may be a bug, please report it to me.
