RedTrans - 小红书专属翻译插件

2025-01-13 翻译 ， 小红书 ， 插件

RedTrans 是一个专门的 Chrome 浏览器扩展程序，可以帮助您翻译小红书网站 (https://www.xiaohongshu.com/) 上的文本内容。

功能特点

  • 可以翻译探索页面的标题、笔记详情页面的标题/内容/评论，以及通知页面的消息
  • 使用大语言模型（LLM）进行翻译，目前支持 OpenAI 和 DeepSeek
  • 简洁轻量，无广告，不追踪，不收集数据

使用方法

安装步骤

  1. 从 https://orrrrz.github.io/downloads/RedTrans-v0.1.zip 下载压缩包并解压（或者您也可以克隆 GitHub 仓库：https://github.com/orrrrz/red-trans）
  2. 打开 Chrome 浏览器，访问 chrome://extensions/ 并切换到开发者模式
  3. 点击”加载已解压的扩展程序”
  4. 选择您解压的文件夹
  5. 选项页面会自动打开，您可以在那里设置您的 API 密钥。您可以从 OpenAI/DeepSeek 网站获取 API 密钥
  6. 完成！

使用说明

将鼠标悬停在标题/评论/通知上，翻译按钮就会显示出来。

祝您使用愉快！

常见问题

如何获取 API 密钥？

  • 对于 DeepSeek，请访问 https://platform.deepseek.com/。如果您还没有注册，请先注册。然后从左侧菜单点击”API Keys”，再点击”Generate API Key”按钮。

  • 对于 OpenAI，请访问 https://platform.openai.com/settings/organization/api-keys。如果您还没有注册，请先注册。点击”Create new secret key”按钮。

翻译按钮没有显示

  • 由于按钮有时可能加载失败，请尝试刷新页面
  • 如果仍然无法显示，可能是程序问题，请向我反馈
